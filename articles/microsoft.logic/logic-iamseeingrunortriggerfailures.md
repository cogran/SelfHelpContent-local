# 出现运行失败或触发器失败

## **建议的步骤**
若要解决最常见问题，请尝试下面的一种或多种方法。

1. 单击状态为**“失败”**的运行或触发器。
1. 查找失败的操作的错误代码，以了解有关问题原因的详细信息。
    - 对于引用错误，请务必检查前面步骤的输出，以确保这些输出与后续步骤的预期输出相匹配。
    - 对于模板无效错误，请根据[工作流定义语言](http://aka.ms/logicappsdocs)检查表达式
