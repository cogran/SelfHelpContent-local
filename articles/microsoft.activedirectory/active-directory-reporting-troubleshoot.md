# <a name="azure-ad-audit-log-and-activity-reports"></a>Azure AD 审核日志和活动报表

## <a name="recommended-steps"></a>**建议的步骤**

操作不会立即显示在审核活动日志中。 从执行操作的时间开始算起，可能需要 30 分钟到 1 小时才能在 Azure 门户中看到审核日志。 等待 30 分钟到 1 小时，然后查看操作是否显示在活动日志中。
