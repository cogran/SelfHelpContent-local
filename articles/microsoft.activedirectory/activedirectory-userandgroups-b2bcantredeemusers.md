# <a name="i-cant-redeem-a-users-invitation"></a>我无法兑换用户的邀请

## <a name="recommended-steps"></a>**建议的步骤**
验证租户是否存在于 Azure AD 中，以及用户对象是否存在于该租户中。 如果用户对象不存在，请将用户对象同步到 Azure AD。
