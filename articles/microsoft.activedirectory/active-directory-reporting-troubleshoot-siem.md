# <a name="can-i-integrate-azure-ad-activity-logs-with-my-siem-system"></a>是否可将 Azure AD 活动日志与 SIEM 系统集成？

## <a name="recommended-steps"></a>**建议的步骤**

可以使用两种受支持的方法来完成此任务：


- 可以使用 Azure 日志集成器来与 SIEM 工具无缝集成

- 可以使用报告图形 API 获取数据，然后使用自己的脚本将其推送到 SIEM 工具
