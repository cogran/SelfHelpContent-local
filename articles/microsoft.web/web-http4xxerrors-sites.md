# 可用性、性能和应用程序问题/HTTP 4xx 错误

## **建议的步骤**
如何排查“错误 403 - 此 Web 应用已停止”问题？ 有 3 种情况可能会导致出现此错误。

1. 站点已达到计费限制，因此已被禁用。
2. 在门户中停止了网站。
3. Azure 网站已达到了适用于免费或共享缩放模式的资源配额限制。
