# 部署/FTP

## **建议的步骤**
问题：使用 FTP 连接网站失败

1. 请检查是否输入了正确的主机名和凭据
2. 请检查防火墙是否阻止了端口。a. FTP 控制连接端口：21。b. FTP 数据连接端口：989、10001-10300。
