# <a name="errors-and-exceptionschannel-disconnected-error"></a>错误和异常/“通道断开连接”错误

## <a name="recommended-steps"></a>**建议的步骤**
* 设备处于脱机状态并且未连接到其平台通知系统时，可能会发生此错误。<br>
* 确认设备处于联机状态，然后重试。<br>
