# <a name="i-need-to-purchase-or-upgrade-my-subscription"></a>我需要购买或升级订阅

## <a name="recommended-steps"></a>**建议的步骤**

* [找出谁是帐户管理员](data-blade:Microsoft_Azure_Billing.SubscriptionPropertiesBlade)。<br>

* 可将免费版 Azure 订阅升级到即用即付版本。<br>
[将免费版订阅升级到即用即付版本](https://docs.microsoft.com/azure/billing/billing-upgrade-azure-subscription/)<br>

* 帐户管理员可将即用即付版订阅更改为其他产品 - 访问帐户中心，选择要更改的订阅，然后单击“切换到其他产品”。<br>
[将订阅更改为其他产品](https://docs.microsoft.com/azure/billing/billing-how-to-switch-azure-offer/)<br>
