# 无法创建 StorSimple 设备管理器
StorSimple 设备管理器仅支持 Microsoft Azure Enterprise 协议和 MSDN 订阅。 不支持即用即付、预付费和 Azure 开放许可。

## **建议的步骤**
* 确认你是否拥有 Microsoft Azure Enterprise 协议或 MSDN 订阅。
