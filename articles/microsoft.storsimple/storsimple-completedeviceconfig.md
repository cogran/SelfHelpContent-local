# <a name="i-am-not-able-to-complete-the-device-configuration-in-the-azure-portal"></a>无法在 Azure 门户中完成设备配置。


## <a name="recommended-steps"></a>**建议的步骤**
1.  在虚拟阵列的本地 Web UI 中，转到 **疑难解答**  >  **诊断测试** ，然后单击 **运行诊断测试** 。 验证是否正确配置了设备，并解决报告的所有问题。
2.  确保在设备上打开了正确的端口。
